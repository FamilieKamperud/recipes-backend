# Recipe API with node and neo4j

## Prerequisites

### Must be installed
- Node.js v6.9.1 and npm v3.10.8
- Neo4j v3.0.6
- Yarn v0.17.0

They must all be available in the command line

## Development

### To start app run

```
./build.sh
```
